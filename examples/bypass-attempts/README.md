# Bypass Attempts Test Cases

This directory contains malicious patterns designed to test and improve cc-audit's detection capabilities.

## Categories

1. **encoding/** - Various encoding techniques to evade detection
2. **multiline/** - Patterns split across multiple lines
3. **indirect/** - Indirect execution via variables, aliases, functions
4. **reverse-shells/** - Various reverse shell implementations
5. **language-specific/** - Language-specific evasion techniques
6. **timing/** - Time-based attacks
7. **obfuscation/** - String manipulation and obfuscation
